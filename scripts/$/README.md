validBraces
================

Выполните, пожалуйста, функцию $ так, чтобы методы addClass и html были вынесены в прототип.
В идеале, они должны быть в прототипе, принадлежащему функции $.

```javascript
$(‘.my-class’).addClass(‘new-class’).html(‘’);
```
