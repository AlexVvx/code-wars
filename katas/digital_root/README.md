digital_root
================

Написать функцию digital_root которая рекурсивно суммирует цифры числа. Если получается двузначное и более,
итерация повторяется, например:

```javascript
// 9 + 4 + 2 = 15 => 1 + 5 = 6
digital_root(942); // 6
```
