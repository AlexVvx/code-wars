####Description:

Write a function which takes an array data of numbers and returns the largest difference in indexes j - i such that data[i] <= data[j]

####Example:

```js
largestDifference([1,2,3]) ; // returns 2, because here j = 2 and i = 0 and 2 - 0 = 2
```
